# Настройка отправки email

## Для настройки реальной отправки email:

1. Создайте файл `.env.local` в корне проекта
2. Добавьте следующие переменные:

```
EMAIL_USER=your-email@gmail.com
EMAIL_PASS=your-app-password
```

## Инструкция по настройке Gmail:

1. **Создайте Gmail аккаунт** или используйте существующий
2. **Включите двухфакторную аутентификацию** в настройках Google аккаунта
3. **Создайте пароль приложения**:
   - Перейдите в [настройки Google аккаунта](https://myaccount.google.com/)
   - Безопасность → Пароли приложений
   - Создайте новый пароль приложения для "Почта"
   - Используйте этот пароль в `EMAIL_PASS`
4. **Замените** `your-email@gmail.com` на ваш реальный Gmail
5. **Перезапустите сервер** после создания `.env.local`

## Безопасность:
- Никогда не коммитьте файл `.env.local` в git
- Используйте только пароли приложений, не основной пароль Gmail
- Ограничьте доступ к переменным окружения

## Тестирование:
После настройки email будет отправляться на указанный адрес с красивым HTML шаблоном.
