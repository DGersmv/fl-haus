# Настройка собственной почты для tashi-ani.ru

## Варианты настройки почты

### 1. Через хостинг-провайдера (reg.ru)
Если домен зарегистрирован на reg.ru, можно использовать их почтовые сервисы:

1. Заходим в панель управления reg.ru
2. Переходим в раздел "Почта" 
3. Создаем почтовый ящик `admin@tashi-ani.ru`
4. Устанавливаем пароль `admin123`
5. Настраиваем SMTP:
   - **Хост**: `smtp.reg.ru` или `mail.tashi-ani.ru`
   - **Порт**: 587 (STARTTLS) или 465 (SSL)
   - **Логин**: `admin@tashi-ani.ru`
   - **Пароль**: `admin123`

### 2. Через Yandex для домена (бесплатно)
1. Заходим в Яндекс.Почта для домена
2. Подключаем домен tashi-ani.ru
3. Создаем ящик admin@tashi-ani.ru
4. Настраиваем SMTP:
   - **Хост**: `smtp.yandex.ru`
   - **Порт**: 587
   - **Логин**: `admin@tashi-ani.ru`
   - **Пароль**: `admin123`

### 3. Через Google Workspace (платно)
1. Регистрируемся в Google Workspace
2. Подключаем домен tashi-ani.ru
3. Создаем ящик admin@tashi-ani.ru
4. Настраиваем SMTP:
   - **Хост**: `smtp.gmail.com`
   - **Порт**: 587
   - **Логин**: `admin@tashi-ani.ru`
   - **Пароль**: пароль приложения

## Настройка переменных окружения

После создания почты, обновляем `.env.local`:

```env
# Email настройки
EMAIL_USER=admin@tashi-ani.ru
EMAIL_PASS=admin123

# Остальные настройки
JWT_SECRET=your-jwt-secret
MASTER_ADMIN_EMAIL=2277277@bk.ru
MASTER_ADMIN_PASSWORD=admin123
DATABASE_URL="file:./dev.db"
```

## Тестирование

После настройки почты, тестируем отправку кода:

1. Заходим на сайт
2. Вводим email клиента (например, 9414141@mail.ru)
3. Нажимаем "Получить код"
4. Проверяем логи сервера на наличие ошибок
5. Проверяем почту клиента на наличие письма

## Преимущества собственной почты

✅ **Надежность** - не зависит от Gmail блокировок
✅ **Брендинг** - письма приходят от admin@tashi-ani.ru
✅ **Контроль** - полный контроль над настройками
✅ **Стабильность** - меньше проблем с блокировками
